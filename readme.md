## 🇬🇧 English
**EmojiFlow** is an educational web app where kids learn programming and cybersecurity basics by building logic flows with emojis.

**Run:** Open `index.html` in any modern browser — no install, no server.

**Modules:** 🧩 Coding | 🔌 IoT Lab | 🔒 Cyber Basics | 🧪 Challenges | ❓ Help

**Levels:** 🐣 Newb → 🧭 Explorer → 💻 Developer

**Privacy:** 100% local — no server — no tracking — works offline

**Features:**
- Drag & drop emoji programming
- Flow execution engine with run/debug/stop
- Simulated IoT devices and MQTT concepts
- Password strength meter and Caesar cipher
- Permission awareness game
- 8 interactive challenges with star system
- Local save/load with JSON export/import
- Trilingual (AR/EN/FR)
- Islamic geometric dark theme
- Responsive (mobile + tablet + desktop)
- Single file, < 83KB, zero dependencies
